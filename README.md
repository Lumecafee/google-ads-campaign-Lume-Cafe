# google-ads-campaign-Lume-Cafe
Google Ads Capstone Project

Overview:
- This project simulates a Google Ads campaign for a premium, design-led café brand focused on quiet luxury and aesthetic experiences.


Objectives: 
- Increase website traffic
- Build brand awareness
- Drive store visits

Strategy:
- Search Ads for high-intent users
- Display Ads for visual storytelling
- Psychology-driven messaging (calm, escape, minimalism)

Project Structure:
- proposal/ → Campaign proposal
- ads/ → Ad creatives & copy
- report/ → Final analysis

Key Learnings:
- Perception plays a key role in premium branding
- Emotional messaging outperforms generic ads
- Keyword specificity improves targeting

Tools Used:
- Google Ads (simulation)
- Canva (for creatives)
- Keyword planning tools
